
🛡️ Vibe Check | Scroll in Peace

Vibe Check is an AI-powered accessibility tool built for the 2026 Hackathon. It uses multimodal Vision AI to scan Reddit text and images in real-time, providing traffic-light safety indicators and automatically blurring intense or triggering content.

📦 Installation



1. Open your browser (Chrome, Brave, or Edge) and go to `chrome://extensions/`.

2. Toggle Developer mode to ON in the top right.

3. Click Load unpacked.

4. Select the `/vibecheck/` folder from this repository.

5. Navigate to Reddit and enjoy a protected browsing experience.

🛠️ How it Works



* Multimodal AI: Uses the Groq Llama 3.2 Vision model to analyze both text and images simultaneously.



* Intelligent Blurring: Automatically applies a frosted-glass overlay to "Intense" posts.



* Shadow DOM Piercing: Custom traversal logic ensures the extension can read content hidden inside modern Reddit's Shadow DOM structure.



* Calm UI: Features a nature-inspired landing page designed to signal safety and peace to the user.
